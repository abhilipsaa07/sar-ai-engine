About This System

This tool assists compliance officers in generating FinCEN-compliant SAR narratives. It uses Retrieval-Augmented Generation (RAG) to pull the most relevant FinCEN regulatory guidance before drafting each narrative. All reports are stored in a persistent audit trail with timestamps.

Stack: FastAPI · SQLite · Anthropic Claude · RAG knowledge base · GitHub Pages
